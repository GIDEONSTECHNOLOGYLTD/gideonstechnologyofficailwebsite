Repair Service API

Welcome to the Repair Service API. Use the following endpoints to interact with our service:

GET /

Get information about the API and available endpoints.

GET /repair/repair

Check if the repair service is online and get the current status.

POST /repair/repair

Submit a repair request to the system.

{
  "device": "iPhone 13",
  "issue": "Cracked screen",
  "customerName": "John Doe",
  "contact": "john@example.com"
}

API Response Format

All API responses are returned in JSON format with the following structure:

{
  "status": "success|error",
  "message": "Human-readable message",
  "data": { ... } (optional)
}
